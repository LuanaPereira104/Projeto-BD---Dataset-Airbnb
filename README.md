# Projeto-BD---Dataset-Airbnb
Projeto desenvolvido na unidade curricular Base de Dados II, Universidade do Minho (2025/2026). Implementação completa do domínio Airbnb em três paradigmas distintos — MySQL, Cassandra e Neo4j — incluindo modelação, carregamento de dados reais (Inside Airbnb) e execução de questões analíticas.
