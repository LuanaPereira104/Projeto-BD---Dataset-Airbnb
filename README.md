# Projeto-BD---Dataset-Airbnb

Projeto desenvolvido na unidade curricular Base de Dados II, Universidade do Minho (2025/2026). Implementação do domínio Airbnb utilizando três tecnologias distintas — MySQL, Cassandra e Neo4j — abrangendo modelação, transformação de dados e execução de questões analíticas. O projeto utiliza dados reais do Inside Airbnb, incluindo alojamentos, anfitriões, disponibilidade diária, avaliações e bairros.

O trabalho inclui a construção do modelo conceptual, a adaptação para modelos relacional, orientado a colunas e orientado a grafos, bem como a implementação das consultas analíticas em SQL, CQL e Cypher. A solução evidencia diferenças entre paradigmas de bases de dados e demonstra como cada tecnologia suporta padrões de procura, preços, disponibilidade e relações entre entidades.
